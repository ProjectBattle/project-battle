# Aviator Cove RV Park

A simple static landing page for Aviator Cove RV Park, built with plain HTML, CSS, and JavaScript.

## Run locally

1. Open the project folder in your browser directly, or start a simple local server from the project root.
2. From the project root, run:
   - Python: `python -m http.server 8000`
   - Node.js: `npx serve .`
3. Visit `http://localhost:8000` in your browser.

## Files

- `index.html` — Page structure and content
- `styles.css` — Responsive styling and aviation-inspired visual design
- `script.js` — Mobile navigation and contact form interaction

## Netlify deployment

This site is ready to deploy to Netlify as a static site.

1. Sign in to Netlify and choose "Add new site".
2. Select "Deploy manually" or connect your Git repository.
3. Set the publish directory to the project root.
4. Deploy the site.

No build step is required.
